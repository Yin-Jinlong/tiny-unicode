用于在`utf8`,`utf16`,`utf32`之间转换。

简单的unicode工具。

此仓库是[color-shell](https://github.com/Yin-Jinlong/color-shell.git)的一个外置依赖库

[Unicode范围](Unicode.md)

## 构建

目标`tiny-unicode`，生成静态库

## 支持

- 判断全宽字符
- 判断Emoji

- `UTF-8`->
    - `UTF-32`(BE)

- `UTF-32`(BE)->
    - `UTF-8`
